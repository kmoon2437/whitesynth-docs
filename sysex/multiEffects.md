# Multi effect(MFX)
Multi effect(이하 MFX)는 지정된 채널에 갖가지 이펙트(distortion,overdrive 등등등 뭐가 많음)를 넣을 수 있는 기능으로, 넣은 이펙트에 따라 정말 별의별 소리를 다 낼 수 있다.
whitesynth에서는 채널별로 16개까지 사용할 수 있으며, 이펙터 번호 순으로 적용된다.

## `0x24` - `0x2F` 파라미터에 대해
`0x24` - `0x2F` 파라미터는 이펙터의 종류에 상관없이 공통으로 들어가는 파라미터이다.

| 파라미터 번호 | 이름 | 값 | 기본값 | 설명 |
| --- | -------- | ----- | --- | ------------ |
| `0x24` | Level | 0 - 16383 | 16383 | 최종 출력 level을 조정한다. |
| `0x25` | (Reserved) | N/A | N/A | 추후 사용할 경우를 대비해 비워놓은 곳 |
| : | : | : | : | : |
| `0x2F` | (Reserved) | N/A | N/A | 추후 사용할 경우를 대비해 비워놓은 곳 |

### None - `00 00 00`

### Amp simulator - `00 10 00`
일렉기타 앰프. 여기서도 후술할 드라이브를 넣을 수 있다.

#### 파라미터

| 파라미터 번호 | 이름 | 값 | 기본값 | 설명 |
| --- | -------- | ----- | --- | ------------ |
| `0x00` | Drive | 0 - 16383 | 0 | drive의 정도를 조정한다. |
| `0x01` | Amp type | Small,Combo,Stack,Tube<br>(0, 1, 2, 3) | Tube (3) | 기타 앰프 종류를 선택한다.[^GuitarAmpType] |

[^GuitarAmpType]: Small = 소형 앰프,Combo = 콤보 앰프,Stack = 스택 앰프,Tube = 진공관 앰프. roland sound canvas와 호환되도록 지원할 때 각각 Small,Built-in,2 stack,3 stack에 대응하도록 구현 예정.

### Overdrive - `00 10 01`
일렉기타의 overdrive 이펙트를 적용한다.

일렉기타용으로 사용할 땐 위의 Amp simulator 앞에 설정해서 사용하는 것이 좋다.

#### 파라미터

| 파라미터 번호 | 이름 | 값 | 기본값 | 설명 |
| --- | -------- | ----- | --- | ------------ |
| `0x00` | Drive | 0 - 16383 | 6144 | drive의 정도를 조정한다. |

### Distortion - `00 10 02`
일렉기타의 distortion 이펙트를 적용한다. 소리를 의도적으로 매우 크게 증폭시킨 다음 클리핑을 하는 이펙트이다.

일렉기타용으로 사용할 땐 위의 Amp simulator 앞에 설정해서 사용하는 것이 좋다.

#### 파라미터

| 파라미터 번호 | 이름 | 값 | 기본값 | 설명 |
| --- | -------- | ----- | --- | ------------ |
| `0x00` | Drive | 0 - 16383 | 9728 | drive의 정도를 조정한다. |

<!-- 컴프레서 (00 23 00) -->

### Compressor - `00 23 00`
흔한 컴프레서. 일정 수준을 넘어선 신호의 레벨을 줄임으로써 레벨의 격차(다이나믹 레인지)를 줄인다.

#### 파라미터

| 파라미터 번호 | 이름 | 값 | 기본값 | 설명 |
| --- | -------- | ----- | --- | ------------ |
| `0x00` | Attack | 0 - 16383 | N/A | threshold를 넘는 소리가 줄어드는 시간을 조정한다. 단위는 밀리초 |
| `0x01` | Makeup gain | 0 - +18<br>(공식: x) | N/A | 출력 gain을 조정한다. |
| `0x02` | Threshold | 0 - 144 | N/A | "특정한 기준점에 맞춤으로써"에서 "특정한 기준점"을 설정한다. 단위는 -dBFS. |
| `0x03` | Ratio | 0.0 - 100.0<br>(공식: 10*x) | N/A | 일반적인 컴프레서에서 Ratio를 "n : 1" 로 표기하는데 이때 n을 설정한다. |
| `0x04` | Release | 0 - 16383 | N/A | threshold에서 멀어진 볼륨이 다시 threshold에 가까워졌을 때 줄이거나 키운 볼륨을 원상복구하는 시간을 조정한다. 단위는 밀리초. |

<!-- 기타 (00 7n nn) -->

### Reverser - `00 70 00`
정해진 주기마다 재생한 소리를 버퍼링하고 그 동안 이전에 버퍼링된 소리를 되감기로 재생한다.

#### 파라미터

| 파라미터 번호 | 이름 | 값 | 기본값 | 설명 |
| --- | -------- | ----- | --- | ------------ |
| `0x00` | BPM | 1 - 16383 | 120 | 템포를 설정한다. |
| `0x01` | BPM decimal fraction | 0 - 9999 | 0 | 템포의 소수점 아래 4자리까지를 설정한다. 예를 들면 3으로 설정한 경우 bpm은 120.0003이 되며, 72로 설정한 경우 bpm은 120.0072가 된다. bpm을 120.3으로 설정하려면 이 파라미터를 3000으로 설정해야 한다. |
| `0x03` | Length | 1 - 16383 | 4 | 버퍼링할 소리의 길이가 몇 박자가 될지를 설정한다. BPM 값과 Beats length 값을 기준으로 소리를 버퍼링하고 되감기할 주기를 결정한다. |
| `0x04` | Exponent | 1 - 16 (온음표 ‐ 65536분음표) | 2 (4분음표) | Length는 그 값에 n분음표의 길이를 곱해서 계산하는데, 이때 n분음표의 n은 2의 k제곱으로 나타낸다. 이때 k를 설정하는 것이 이 파라미터이다. 기본값은 2. 즉 2의 제곱 = 4분음표다. |
| `0x05` | Mix | 0 - 16383 | 16383 | 입/출력 신호를 얼마나 섞을지 결정한다. 0 또는 1이면 입력 신호만, 16383이면 출력 신호만 출력되며, 8192면 반반으로 섞어 출력된다. |
| `0x0F` | Reset buffer | 0 | - | 버퍼를 초기화한다. 박자가 안 맞을 경우 보정하는 용도이며, 파라미터로 작용하지는 않지만 일단 값을 뭐라도 주긴 줘야 한다. |

### Bit crusher - `00 70 01`
소리의 bit depth와 sample rate를 의도적으로 낮춘다(즉, 음질을 의도적으로 열화시킨다).

#### 파라미터

| 파라미터 번호 | 이름 | 값 | 기본값 | 설명 |
| --- | -------- | ----- | --- | ------------ |
| `0x00` | Bit depth | 1 - 16 | 8 | bit depth를 설정한다. 평범한 음질은 보통 16비트 정수이기 때문에 최대값은 16비트이다. |
| `0x01` | Frequency | 1 - 100 | 5 | sample rate를 몇 배 줄일지 설정한다. 값이 커질수록 음질이 낮아진다. |

### Monoifier - `00 70 02`
오디오의 좌/우를 합쳐 mono 형식으로 만든다.

#### 파라미터

| 파라미터 번호 | 이름 | 값 | 기본값 | 설명 |
| --- | -------- | ----- | --- | ------------ |
| `0x00` | Pan | L8191(1) - Center(8192) - R8191(16383) | Center(8192) | mono로 합친 뒤 출력될 소리의 pan을 조절한다. |
